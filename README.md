### 映射

F1 帮助文档  
F2 显示/关闭行号，默认为显示  
<F4> 重新载入当前文件  
<F5> 运行文件（如果可以的话）  
C-F5 编译文件（如果可以的话）  
S-F5 编译文件（如果可以的话）  
(C-F5 和<S-F5>暂时不确定用哪一个比较顺手，所以两个都设置为了编译  )  
F7 浅色模式  
F8 深色模式  
F12 打开配置文件  


### 代码适配

目前只适配了go和python的基础运行，后续再进行更新


### 使用方式
1. 替换vimrc或者添加vimrc内容
2. 打开init.vim文件
3. 在末行模式运行:source
4. 等待插件加载完成
